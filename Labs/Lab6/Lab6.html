This is my first draft of my logo, my logo in black and white, and in color.

What exactly is a Favicon

A favicon is the image at the top next to the url. It's the logo of your website. It is used to distiguish between other titles that are similiar.


How are favicons created?

Favicons are created by downloading an image and imputting them into the code.


How did I create the favicon for this site?

I created the favicon for this page on code.org. What I did there was create a 32x32 image. For each pixel I added the binary code for the color I wanted. I used two colors for my logo a light blue and a light purple. The light blue binary code was 110101 and the binary code for the light purple was used is 010111. The last step was to just download the image and input into my head part of code.
